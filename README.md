# OdinJSProject_2

## Project Overview

This project is a browser-based application that combines the functionalities of a sketchpad and an Etch-A-Sketch. Users can create and manipulate drawings in a simple and intuitive interface.


## Technologies Used

- HTML5
- CSS3
- JavaScript

## What I learned 
- Creating grid of square divs 
- Handling container size based on user input
- Usage of mousedown, mousemove and mouseup events
